# Publish This Privacy Policy In 5 Minutes

## 1) Create a new GitHub repo
Create a public repo, for example: `dj-job-manager-privacy`

## 2) Upload files
Upload:
- `index.html`

## 3) Enable GitHub Pages
- Go to **Settings -> Pages**
- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/ (root)**

## 4) Copy the live URL
It will look like:
- `https://<your-username>.github.io/dj-job-manager-privacy/`

## 5) Paste this URL in Chrome Web Store
Use it in the Privacy Policy URL field.

## Before publishing
Replace `YOUR_SUPPORT_EMAIL` in `index.html` with your real support email.

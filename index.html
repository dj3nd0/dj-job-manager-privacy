Privacy Policy for DJ Job Manager

Effective date: May 20, 2026

DJ Job Manager is a browser extension for extracting job-posting data selected by the user and sending it to a user-configured API endpoint.

Data We Process

  • Website content from pages where the user triggers extraction.
  • Current page URL/path used for extraction and matching.
  • User configuration stored in Chrome sync storage:
  • API endpoint
  • Admin URL
  • Extraction configuration

How We Use Data

Data is processed only to provide the extension's single purpose: extracting job data and supporting personal job-tracking workflows.

Data Sharing

  • Data is sent only to the API endpoint configured by the user.
  • We do not sell user data.
  • We do not use or transfer user data for advertising personalization.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.

Security

Data is transmitted through secure connections supported by the configured endpoint.

Retention

  • Browser-stored settings remain until changed by the user or removed with the extension.
  • Server-side retention is controlled by the user-configured backend.

User Control

  • Users control when extraction runs.
  • Users control where extracted data is sent.
  • Users can clear extension data via browser settings and uninstall at any time.
